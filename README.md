# API de Filmes

## 📌 Tema
Sistema de gerenciamento de filmes

## ⚙️ Funcionalidades
- Listar filmes
- Filtrar filmes assistidos

## 🗄️ Dados armazenados
- Título
- Status (Assistido / Não Assistido)

## 🚀 Como rodar
1. Instale as dependências: `pip install -r requirements.txt`
2. Rode o servidor: `python manage.py runserver`
